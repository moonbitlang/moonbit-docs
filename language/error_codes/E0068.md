# E0068

Warning name: `declaration_unimplemented`

Declaration is unimplemented.

This warning is emitted when a declaration is left unimplemented. Provide an
implementation or remove the declaration.

## Erroneous example

```moonbit
declare fn f() -> Unit
```

## Suggestion

Provide an implementation:

```moonbit
fn f() -> Unit {
  println("Hello")
}
```

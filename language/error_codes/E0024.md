# E0024

Warning name: `unused_error_type`

The error type of this function is never used.

## Erroneous example

```moonbit
///|
fn greet() -> Unit raise {
  println("Hello, world!")
}
```

## Suggestion

Remove the error type:

```moonbit
///|
fn greet() -> Unit {
  println("Hello, world!")
}
```

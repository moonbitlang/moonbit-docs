# E4057

The constructor is duplicate.

## Erroneous example

```moonbit
///|
enum E {
  A
  A(Int)
}
```

## Suggestion

Rename the constructor to a different name.

```moonbit
///|
priv enum E {
  A
  B(Int)
}

///|
test {
  let value = E::B(1)
  ignore(E::A)
  match value {
    A => inspect("A")
    B(n) => inspect(n, content="1")
  }
}
```

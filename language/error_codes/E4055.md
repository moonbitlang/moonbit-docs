# E4055

Field is already declared.

## Erroneous example

```moonbit
///|
struct A {
  a : Int
  a : Double
}
```

## Suggestion

Rename the field to a different name.

```moonbit
///|
priv struct A {
  a : Int
  b : Double
}

///|
test {
  let value : A = { a: 1, b: 2.0 }
  inspect(value.a, content="1")
  inspect(value.b, content="2")
}
```

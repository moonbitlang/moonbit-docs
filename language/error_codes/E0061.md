# E0061

Warning name: `duplicate_test`

Duplicate test name.

This warning is emitted when two (or more) tests share the same name.

## Erroneous example

```moonbit
test "dup" {
  inspect(1, content="1")
}

test "dup" {
  inspect(2, content="2")
}

```

## Suggestion

Rename one of the tests:

```moonbit
test "dup" {
  inspect(1, content="1")
}

test "dup2" {
  inspect(2, content="2")
}

```

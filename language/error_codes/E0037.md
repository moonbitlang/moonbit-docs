# E0037

Warning name: `unused_loop_label`

The loop label name is never used.

## Erroneous example

```moonbit
///|
fn main {
  let mut sum = 0
  let input = [1, 2, 3]
  read~: for item in input {
    sum += item
  }
  println(sum)
}
```

## Suggestion

Remove the unused loop label.

```moonbit
///|
fn main {
  let mut sum = 0
  let input = [1, 2, 3]
  for item in input {
    sum += item
  }
  println(sum)
}
```

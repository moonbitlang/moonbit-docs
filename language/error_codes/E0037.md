# E0037

Warning name: `unused_loop_label`

The loop label name is never used.

## Erroneous example

```moonbit
///|
fn main {
  let mut sum = 0
  let input = [1, 2, 3]
  read~: loop input[:] {
    [] => break
    [head, .. tail] => {
      sum += head
      continue tail
    }
  }
  println(sum)
}
```

## Suggestion

Remove the unused loop label.

```moonbit
///|
fn main {
  let mut sum = 0
  let input = [1, 2, 3]
  loop input[:] {
    [] => break
    [head, .. tail] => {
      sum += head
      continue tail
    }
  }
  println(sum)
}
```

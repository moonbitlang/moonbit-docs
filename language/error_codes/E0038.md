# E0038

Warning name: `missing_invariant`

## Erroneous example

The loop does not have a `proof_invariant` clause.

```moonbit
///|
fn fib(n : Int) -> UInt64 {
  for i = 0, a = 0UL, b = 1UL; i < n; i = i + 1, a = b, b = a + b {

  } else {
    b
  }
}
```

## Suggestion

Add some proof invariants.

```moonbit
///|
fn fib(n : Int) -> UInt64 {
  for i = 0, a = 0UL, b = 1UL; i < n; i = i + 1, a = b, b = a + b {

  } nobreak {
    b
  } where {
    proof_invariant: a <= b,
  }
}
```

# E4204

Compiler diagnostic name: `loop_nobreak_not_supported`.

The deprecated functional `loop` expression does not support a `nobreak` block.
`nobreak` is supported on ordinary loop forms such as `while` and `for`, but a
functional `loop` should return through an explicit `break` instead.

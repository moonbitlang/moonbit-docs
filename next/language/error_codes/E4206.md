# E4206

Compiler diagnostic name: `must_implement_one`.

Traits can use `#must_implement_one` to require each implementation to provide
at least one method explicitly. When the attribute lists method names, an
implementation must define at least one of those listed methods; inherited
default implementations are not enough.

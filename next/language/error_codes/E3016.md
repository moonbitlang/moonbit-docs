# E3016

Unexpected `~` in argument. Did you mean `label=pattern` or `label~`?

## Erroneous example

```{literalinclude} /sources/error_codes/3016_error/top.mbt
:language: moonbit
```

## Suggestion

Replace `b~` with `b=` or `b~` as appropriate.

```{literalinclude} /sources/error_codes/3016_fixed/top.mbt
:language: moonbit
```

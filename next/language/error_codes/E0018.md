# E0018

Warning name: `useless_loop`

There is no `continue` in this loop expression, so `loop` is useless here.

This warning occurs when using a `loop` expression that contains no `continue`
statement.

```{note}
This warning is tied to the deprecated `loop` syntax. Current MoonBit also
reports `deprecated_syntax` for that construct, so this compatibility page has
no checked source example.
```

In legacy code, a `loop` without any `continue` statements is equivalent to a
block expression that will be executed only once, making the `loop` keyword
redundant.

## Suggestion

Use a simpler control flow construct like `match` or `if` when the code does not
need to continue the loop:

```moonbit
let count = 2
if count >= 0 {
  println(count)
}
```

If the code needs to carry state across iterations, rewrite it with a current
`for` loop form and use `continue` only when the loop should proceed to another
iteration.

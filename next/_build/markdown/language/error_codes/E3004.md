# E3004

Missing parameters list. Add `()` after the name of the function if it takes 0
parameter.

## Erroneous example

```moonbit
///|
/// Error: Missing parameters list. Add `()` if function `greet` has 0 parameter.
pub fn greet -> Unit {
  println("Hello, world!")
}
```

## Suggestion

Add `()` after the function name.

```moonbit
///|
pub fn greet() -> Unit {
  println("Hello, world!")
}
```

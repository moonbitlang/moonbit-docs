# 循环

本示例展示了使用 for 循环和 while 循环遍历数组。

## for 循环表达式

for 循环类似于 C 风格的 for 循环：

```
for 初始化; 条件; 增量 {
    // 循环体
}
```

循环开始前执行初始化部分。每次迭代时：
1. 检查条件表达式
2. 若条件为真则执行循环体
3. 执行增量表达式
4. 重复上述步骤直到条件为假

MoonBit 的 for 循环比 C 风格更灵活，后续章节将详细说明。

## while 循环表达式

while 循环同样类似 C 风格：

```
while 条件 {
    // 循环体
}
```

执行流程：
1. 检查条件表达式
2. 若条件为真则执行循环体
3. 重复上述步骤直到条件为假

MoonBit 也支持在循环中使用`continue`和`break`语句。
